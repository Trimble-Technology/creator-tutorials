# UV nodes


UV nodes are nodes that modify, interact, or apply UV operations to NURBS curves and surfaces.

See <a href="/concepts/GeneralConcepts/nurbsCurve.md" target="_blank">NURBS curve</a> or <a href="/concepts/GeneralConcepts/nurbsSurface.md" target="_blank">NURBS surface</a> for more information pertaining to each primitive type.
