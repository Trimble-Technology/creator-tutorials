# Ceiling

**_Rounds up to the nearest integer._**

---


#### Inputs

* _value_

  * The value to round up.


#### Outputs

* _result_

  * The rounded up value.

* _result list_

  * The list of rounded up values.


### Note(s)

* Other names for this node include: Round up, and Roof.
