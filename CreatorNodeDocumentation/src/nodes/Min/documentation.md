# Minimum

**_Finds the minimum value amongst inputs._**

---


#### Inputs

* _value 1_

  * The first value to compare against.

* _value 2_

  * The second value to compare against.


#### Outputs

* _result_

  * The minimum value.

* _result list_

  * The list of minimum values.
