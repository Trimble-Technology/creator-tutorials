# Clamp

**_Clamps a value between a minimum and a maximum._**

---


#### Inputs

* _value_

  * The value to clamp.

* _min_

  * The minimum of the clamped range.

* _max_

  * The maximum of the clamped range.


#### Outputs

* _result_

  * The clamped value.

* _result list_

  * The list of clamped values.
