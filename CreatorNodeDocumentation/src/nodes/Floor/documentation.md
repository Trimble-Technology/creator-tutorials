# Floor

**_Rounds down to the nearest integer._**

---


#### Inputs

* _value_

  * The value to round down.


#### Outputs

* _result_

  * The rounded down value.

* _result list_

  * The list of rounded down values.


### Note(s)

* Other names for this node include: Round down.
