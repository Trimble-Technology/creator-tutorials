# Normalize vector

**_Normalizes vectors._**

---


#### Inputs

* _vector_

  * The vector value to normalize.

* _reverse_

  * Sets whether to reverse the resulting vector or not.


#### Outputs

* _result_

  * The normalized vector.

* _result list_

  * The list of results of normalized vectors.
