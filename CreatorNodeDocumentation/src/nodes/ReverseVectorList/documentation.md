# Reverse vector list

**_Reverses a vector list._**

---


#### Inputs

* _list_

  * The list of vector values to reverse.


#### Outputs

* _list_

  * The reversed list of vector values.


### Note(s)

* Other names for this node include: ReverseVectorList.
