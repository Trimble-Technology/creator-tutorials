# Maximum

**_Finds the maximum value amongst inputs._**

---


#### Inputs

* _value 1_

  * The first value to compare against.

* _value 2_

  * The second value to compare against.


#### Outputs

* _result_

  * The maximum value.

* _result list_

  * The list of maximum values.
