# Not

**_Negates the boolean input._**

---


#### Inputs

* _value_

  * The boolean value to negate.


#### Outputs

* _result_

  * The negated boolean value.

* _result list_

  * The list of negated boolean values


### Note(s)

* Other names for this node include: Invert boolean, and !.
